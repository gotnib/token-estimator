Features
Everything you need to
write leaner prompts
TokenLens breaks down exactly where your tokens go and shows you how to get the same results for less.
📊
Token breakdown
See exactly which parts of your prompt consume the most tokens — core instruction, context, examples, filler, and more — broken down with a visual bar chart.
💰
Cost estimator
Know your exact cost per call before you run it. We calculate against Anthropic's Sonnet pricing so you can budget your API usage with confidence.
✨
Efficient rewrite
Claude rewrites your prompt to say the same thing in fewer tokens. Not paraphrased nonsense — a genuinely tighter version that performs just as well.
💡
Actionable reasons
Don't just see a shorter prompt — understand why it's better. We explain each optimization so you can apply the same thinking to future prompts yourself.
⚡
Instant results
No waiting around. Paste your prompt, hit analyze, and get the full breakdown in under 3 seconds — token count, cost, rewrite, and explanation all at once.
🔒
Secure by design
Your prompts go directly to Anthropic's API through our server. Nothing is stored, logged, or used for training. Your ideas stay yours.
How it works
Three steps to smarter prompts
01
Paste your prompt
Copy any prompt you're working on — a system prompt, a user message, a template. The longer and wordier, the more we can save you.
02
Claude analyzes it
Our API sends your prompt to Claude, which breaks it down by category, estimates token count and cost, and drafts a leaner version.
03
Apply and save
Review the breakdown, copy the optimized prompt, and understand the reasoning. Apply the same principles to every prompt you write going forward.
