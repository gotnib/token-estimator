Installation instructions
Chrome / Chromium / Arc / Brave
1
Download the Chrome .zip file above and extract the folder to somewhere you'll remember.
2
Open a new tab and navigate to chrome://extensions
3
Toggle Developer mode on using the switch in the top-right corner.
4
Click Load unpacked and select the extracted tokenlens-v2 folder.
5
The TL icon will appear in your toolbar. Pin it for easy access and log in with your TokenLens account.
Firefox
1
Download the Firefox .zip file above — do not extract it.
2
Open a new tab and navigate to about:addons
3
Click the gear icon ⚙️ and select Install Add-on From File…
4
Select the downloaded ff-tokenlens-extension-v2.zip file and click Add.
5
The TL icon will appear in your toolbar. Log in with your TokenLens account to get started.
